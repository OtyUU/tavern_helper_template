# Резюме: Автоматическое горизонтальное движение камеры (pan)

## Что было сделано

Реализована функция автоматического центрирования камеры по горизонтали в `src/renpy-player`. Теперь камера автоматически держит все видимые спрайты в центре кадра без необходимости явных команд.

## Пример использования

### До изменений
```renpy
show chinami at center
"Я в центре"
show mirai at left
"Mirai слева, но есть пустое пространство справа"
```
Результат: Chinami и Mirai не центрированы, пустое пространство справа.

### После изменений
```renpy
show chinami at center
"Я в центре"
show mirai at left
"Камера автоматически сдвинется, чтобы мы обе были в центре кадра"
```
Результат: Камера автоматически панорамирует, чтобы оба персонажа были в центре.

### Алгоритм

```typescript
// 1. Получить позиции всех спрайтов (в %)
const spriteXPositions = sprites.map(sprite => getSpriteAnchorXPct(sprite.position));

// 2. Найти границы
const minX = Math.min(...spriteXPositions);
const maxX = Math.max(...spriteXPositions);

// 3. Вычислить центр спрайтов
const spritesCenter = (minX + maxX) / 2;

// 4. Вычислить смещение для центрирования
const panOffset = 50 - spritesCenter; // 50% = центр сцены
```

### Приоритет

1. **Ручной pan** (если `panXPct` указан в `PlayerCameraIntent`) - используется он
2. **Автоматический pan** (если ручной не указан) - вычисляется автоматически
3. **Нет спрайтов** - pan = 0

## Новые настройки

### Auto-pan Sensitivity (Чувствительность автоматического pan)
- **Диапазон:** 0.0 - 1.0
- **По умолчанию:** 1.0
- **Где:** Settings → Camera presets → Auto-pan sensitivity
- **Описание:** Контролирует силу автоматического центрирования
  - `0.0` = отключено (камера не двигается)
  - `0.5` = половинное центрирование
  - `1.0` = полное центрирование

### Background Parallax (Параллакс фона)
- **Диапазон:** 0.0 - 1.0
- **По умолчанию:** 0.85
- **Где:** Settings → Camera presets → Background parallax
- **Описание:** Контролирует, насколько фон следует за движением камеры
  - `0.0` = фон статичен (максимальный параллакс)
  - `0.5` = фон двигается на 50%
  - `1.0` = фон двигается полностью (нет параллакса)
