# Тест автоматического горизонтального pan камеры

## Описание изменений

Добавлена функция автоматического центрирования камеры по горизонтали (pan), которая держит все видимые спрайты в центре кадра.

### Новые возможности:

1. **Автоматическое центрирование**: Камера автоматически панорамирует, чтобы держать всех персонажей в центре
2. **Настройка чувствительности**: Контролируйте силу автоматического центрирования (0.0 - 1.0)
3. **Настройка параллакса**: Контролируйте, насколько фон следует за камерой (0.0 - 1.0)

### Как это работает:

1. **Автоматический расчет**: Система вычисляет центр между крайним левым и крайним правым спрайтом
2. **Чувствительность**: Применяется множитель чувствительности (0.0 = отключено, 1.0 = полное центрирование)
3. **Плавное движение**: Камера плавно перемещается, чтобы центрировать все спрайты
4. **Параллакс**: Фон двигается с настраиваемой скоростью относительно спрайтов
5. **Приоритет ручного управления**: Если в команде `camera` явно указан `panXPct`, используется он вместо автоматического

### Настройки:

**Auto-pan Sensitivity** (Чувствительность автоматического pan)
- Где: Settings → Camera presets → Auto-pan sensitivity
- Диапазон: 0.0 - 1.0
- По умолчанию: 1.0
- 0.0 = отключено, 1.0 = полное центрирование

**Background Parallax** (Параллакс фона)
- Где: Settings → Camera presets → Background parallax
- Диапазон: 0.0 - 1.0
- По умолчанию: 0.85
- 0.0 = фон статичен (максимальный параллакс), 1.0 = фон двигается полностью

### Пример использования:

```renpy
scene bg classroom

show chinami at center
"Я в центре"

show mirai at left
"Теперь камера автоматически сдвинется, чтобы мы обе были в центре кадра"

show sakura at right
"Камера снова подстроится под трёх персонажей"

hide mirai
"Камера вернётся к центрированию chinami и sakura"
```

### Технические детали:

**Изменённые файлы**:
- `src/renpy-player/settings.ts` - добавлена настройка `autoPanSensitivity`
- `src/renpy-player/useRenpyPlayerController.ts` - реализован автоматический pan
- `src/renpy-player/SettingsPanel.vue` - добавлен UI для настройки
- `src/renpy-player/context.md` - обновлена документация

**Добавлено**:
- `autoPanXPct` - computed свойство, которое вычисляет оптимальный pan на основе позиций спрайтов
- `cameraPanXPct` - computed свойство, которое выбирает между ручным и автоматическим pan
- Обновлена `cameraPanXPx` для использования нового `cameraPanXPct`
- Применяется чувствительность к автоматическому pan

**Алгоритм**:
1. Получить все видимые спрайты
2. Найти минимальную и максимальную X-позицию (в процентах от ширины сцены)
3. Вычислить центр: `(minX + maxX) / 2`
4. Вычислить смещение: `50% - центр_спрайтов`
5. Применить чувствительность: `смещение * autoPanSensitivity`
6. Применить параллакс к фону: `смещение * bgPanParallax`

### Тестирование:

Чтобы протестировать:
1. Откройте renpy-player в браузере
2. Используйте скрипт с несколькими персонажами на разных позициях
3. Наблюдайте, как камера автоматически центрирует их
4. Проверьте диагностическую панель - там будет показан текущий panX
5. Попробуйте разные значения чувствительности и параллакса в настройках

### Примеры с разными настройками:

**Полное центрирование (по умолчанию)**
```
Auto-pan sensitivity: 1.0
Background parallax: 0.85
```

**Мягкое центрирование**
```
Auto-pan sensitivity: 0.5
Background parallax: 0.85
```

**Отключённое центрирование**
```
Auto-pan sensitivity: 0.0
Background parallax: 0.85 (не важно)
```

**Максимальный параллакс**
```
Auto-pan sensitivity: 1.0
Background parallax: 0.0
```

**Без параллакса**
```
Auto-pan sensitivity: 1.0
Background parallax: 1.0
```

### Диагностика:

В диагностической панели теперь отображается:
- `panX: X.X% (XXXpx)` - текущее горизонтальное смещение камеры
- `panY: XXXpx` - вертикальное смещение
