# Визуальная диаграмма работы автоматического pan

## Концепция

```
┌─────────────────────────────────────────────────────────────┐
│                        СЦЕНА (100%)                          │
│                                                              │
│                    Центр сцены (50%)                         │
│                           ↓                                  │
│                           │                                  │
└─────────────────────────────────────────────────────────────┘
```

## Пример 1: Один спрайт в центре

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│                          [👤]                                │
│                        chinami                               │
│                        (50%)                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘

minX = 50%, maxX = 50%
spritesCenter = (50 + 50) / 2 = 50%
panOffset = 50 - 50 = 0%
✓ Камера не двигается
```

## Пример 2: Два спрайта симметрично

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│         [👤]                              [👤]               │
│        mirai                            sakura               │
│        (30%)                            (70%)                │
│                                                              │
└─────────────────────────────────────────────────────────────┘

minX = 30%, maxX = 70%
spritesCenter = (30 + 70) / 2 = 50%
panOffset = 50 - 50 = 0%
✓ Камера не двигается (уже центрированы)
```

## Пример 3: Два спрайта слева

```
ИСХОДНОЕ ПОЛОЖЕНИЕ:
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│    [👤]        [👤]                                          │
│   mirai      chinami                                         │
│   (30%)       (50%)                                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘

minX = 30%, maxX = 50%
spritesCenter = (30 + 50) / 2 = 40%
panOffset = 50 - 40 = +10%

ПОСЛЕ АВТОМАТИЧЕСКОГО PAN (+10%):
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│              [👤]        [👤]                                │
│             mirai      chinami                               │
│            (40%)       (60%)                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
✓ Камера сдвинулась вправо на 10%, спрайты теперь центрированы
```

## Пример 6: Динамическое изменение

```
ШАГ 1: Один спрайт в центре
┌─────────────────────────────────────────────────────────────┐
│                          [👤]                                │
│                        chinami (50%)                         │
└─────────────────────────────────────────────────────────────┘
panOffset = 0%

ШАГ 2: Добавляем спрайт слева
┌─────────────────────────────────────────────────────────────┐
│    [👤]        [👤]                                          │
│   mirai      chinami                                         │
│   (30%)       (50%)                                          │
└─────────────────────────────────────────────────────────────┘
panOffset = +10% → Камера плавно сдвигается вправо

ШАГ 3: После pan
┌─────────────────────────────────────────────────────────────┐
│              [👤]        [👤]                                │
│             mirai      chinami                               │
│            (40%)       (60%)                                 │
└─────────────────────────────────────────────────────────────┘
✓ Спрайты центрированы

ШАГ 4: Добавляем спрайт справа
┌─────────────────────────────────────────────────────────────┐
│              [👤]        [👤]        [👤]                    │
│             mirai      chinami     sakura                    │
│            (40%)       (60%)       (80%)                     │
└─────────────────────────────────────────────────────────────┘
panOffset = -10% → Камера плавно сдвигается влево

ШАГ 5: После pan
┌─────────────────────────────────────────────────────────────┐
│         [👤]        [👤]        [👤]                         │
│        mirai      chinami     sakura                         │
│       (30%)       (50%)       (70%)                          │
└─────────────────────────────────────────────────────────────┘
✓ Все три спрайта центрированы
```

## Формула

```
Для N спрайтов на позициях [x₁, x₂, ..., xₙ]:

1. minX = min(x₁, x₂, ..., xₙ)
2. maxX = max(x₁, x₂, ..., xₙ)
3. spritesCenter = (minX + maxX) / 2
4. panOffset = 50% - spritesCenter

Где:
- 50% = центр сцены
- panOffset > 0 → камера двигается вправо (спрайты влево)
- panOffset < 0 → камера двигается влево (спрайты вправо)
- panOffset = 0 → камера не двигается
```

## Позиции спрайтов (по умолчанию)

```
farleft   left    midleft  center  midright  right   farright
  10%      30%      40%      50%      60%      70%      90%
   │        │        │        │        │        │        │
   ▼        ▼        ▼        ▼        ▼        ▼        ▼
┌──────────────────────────────────────────────────────────┐
│  [👤]    [👤]    [👤]    [👤]    [👤]    [👤]    [👤]   │
└──────────────────────────────────────────────────────────┘
```

Примечание: Точные значения зависят от настроек `spriteCenterX`, `spriteFarSpacing`, `spriteSideSpacing`, `spriteMidSpacing` в settings.
