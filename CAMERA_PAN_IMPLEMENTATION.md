# Реализация автоматического горизонтального pan камеры

## Краткое описание

Добавлена функция автоматического центрирования камеры по горизонтали, которая держит все видимые спрайты в центре кадра без необходимости явных команд. Теперь включает настройки чувствительности и параллакса.

```typescript
// Автоматический расчёт pan на основе позиций спрайтов
const autoPanXPct = computed(() => {
  const sprites = displayedSprites.value ?? [];
  if (sprites.length === 0) return 0;

  const spriteXPositions = sprites.map(sprite => getSpriteAnchorXPct(sprite.position));
  const minX = Math.min(...spriteXPositions);
  const maxX = Math.max(...spriteXPositions);
  const spritesCenter = (minX + maxX) / 2;
  const stageCenter = 50;
  const panOffset = stageCenter - spritesCenter;
  
  // Применяем чувствительность
  const sensitivity = settings.value.autoPanSensitivity;
  return panOffset * sensitivity;
});

// Выбор между ручным и автоматическим pan
const cameraPanXPct = computed(() => {
  const manualPct = displayedCamera.value?.panXPct;
  if (manualPct !== undefined) {
    return manualPct;
  }
  return autoPanXPct.value;
});
```

## Настройки

### Auto-pan Sensitivity (Чувствительность автоматического pan)

**Диапазон:** 0.0 - 1.0  
**По умолчанию:** 1.0  
**Расположение:** Settings → Camera presets → Auto-pan sensitivity

**Значения:**
- `0.0` - Автоматический pan полностью отключён (камера не двигается)
- `0.5` - Камера центрирует спрайты на 50% (половина необходимого смещения)
- `1.0` - Полное автоматическое центрирование (по умолчанию)

**Примеры:**

```typescript
// Sensitivity = 1.0 (полное центрирование)
// Спрайты на 30% и 50%, центр = 40%
// panOffset = 50 - 40 = 10%
// Результат: камера сдвигается на 10%

// Sensitivity = 0.5 (половинное центрирование)
// panOffset = 10% * 0.5 = 5%
// Результат: камера сдвигается на 5%

// Sensitivity = 0.0 (отключено)
// panOffset = 10% * 0.0 = 0%
// Результат: камера не двигается
```

### Background Parallax (Параллакс фона)

**Диапазон:** 0.0 - 1.0  
**По умолчанию:** 0.85  
**Расположение:** Settings → Camera presets → Background parallax

**Значения:**
- `0.0` - Фон не двигается (максимальный параллакс)
- `0.5` - Фон двигается на 50% от движения спрайтов
- `0.85` - Фон почти следует за спрайтами (по умолчанию)
- `1.0` - Фон двигается вместе со спрайтами (нет параллакса)

**Примеры:**

```typescript
// Автоматический pan = 10%
// Parallax = 0.85
// Фон сдвигается на: 10% * 0.85 = 8.5%
// Спрайты сдвигаются на: 10%
// Эффект: лёгкий параллакс

// Автоматический pan = 10%
// Parallax = 0.5
// Фон сдвигается на: 10% * 0.5 = 5%
// Спрайты сдвигаются на: 10%
// Эффект: заметный параллакс

// Автоматический pan = 10%
// Parallax = 0.0
// Фон сдвигается на: 10% * 0.0 = 0%
// Спрайты сдвигаются на: 10%
// Эффект: максимальный параллакс (фон статичен)
```

## Как это работает

### Алгоритм

1. **Получение спрайтов**: Берём все видимые спрайты из `displayedSprites`
2. **Расчёт позиций**: Для каждого спрайта получаем его X-позицию в процентах от ширины сцены
3. **Поиск границ**: Находим минимальную (`minX`) и максимальную (`maxX`) позиции
4. **Центр спрайтов**: Вычисляем центральную точку: `(minX + maxX) / 2`
5. **Расчёт смещения**: Вычисляем, насколько нужно сдвинуть камеру: `50% - центр_спрайтов`
6. **Применение чувствительности**: Умножаем на `autoPanSensitivity`: `panOffset * sensitivity`
7. **Применение параллакса**: Для фона умножаем на `bgPanParallax`: `panOffset * parallax`

### Примеры

**Один спрайт в центре (50%):**
- `minX = 50`, `maxX = 50`
- `spritesCenter = 50`
- `panOffset = 50 - 50 = 0` ✓ Нет смещения

**Два спрайта: left (30%) и right (70%):**
- `minX = 30`, `maxX = 70`
- `spritesCenter = 50`
- `panOffset = 50 - 50 = 0` ✓ Уже центрированы

**Два спрайта: center (50%) и left (30%) с sensitivity = 1.0:**
- `minX = 30`, `maxX = 50`
- `spritesCenter = 40`
- `panOffset = 50 - 40 = 10`
- `finalPan = 10 * 1.0 = 10%` ✓ Камера сдвигается вправо на 10%

**Два спрайта: center (50%) и left (30%) с sensitivity = 0.5:**
- `minX = 30`, `maxX = 50`
- `spritesCenter = 40`
- `panOffset = 50 - 40 = 10`
- `finalPan = 10 * 0.5 = 5%` ✓ Камера сдвигается вправо на 5%

**Два спрайта: farleft (10%) и left (30%) с parallax = 0.85:**
- `minX = 10`, `maxX = 30`
- `spritesCenter = 20`
- `panOffset = 50 - 20 = 30`
- Спрайты сдвигаются на: `30%`
- Фон сдвигается на: `30% * 0.85 = 25.5%`

## Приоритет

1. **Ручной pan** (`panXPct` в `PlayerCameraIntent`): Если указан, используется он
2. **Автоматический pan с чувствительностью**: Если ручной не указан, используется `autoPanXPct * autoPanSensitivity`
3. **Нет спрайтов**: Если спрайтов нет, pan = 0

## Технические детали

### Производительность
- Все вычисления выполняются в computed свойствах Vue
- Пересчёт происходит только при изменении `displayedSprites` или настроек
- Нет дополнительных watchers или side effects

### Совместимость
- Обратная совместимость: старые скрипты работают как раньше
- Прямая совместимость: готово к добавлению ручного управления pan
- Настройки сохраняются в переменных скрипта

### Ограничения
- Не учитывает размер спрайтов (только позиции)
- Не учитывает границы фона (может панорамировать за пределы)
- Простой линейный алгоритм (без весов или приоритетов)
